# Sunbird AI Pipeline

A Streamlit app that transcribes, summarises, translates and speaks text using Sunbird AI APIs.

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your token to `.env`: `SUNBIRD_API_TOKEN=your_token`
4. Run: `streamlit run app.py`
